# Живые обои для фона Telegram-чатов

Набор анимированных фонов (живых обоев), предназначенных для установки в Telegram Desktop и мобильные приложения.

## Что здесь есть
- Анимированные обои в формате `.mp4` (или `.webm`)
- Инструкция по установке
- Возможные темы: космос, природа, абстракция, техно

## Как установить
1. Скачайте видеофайл обоев.
2. В Telegram:
   - **Настройки** → **Чат-фон** → **Добавить новый фон**
   - Выберите видеофайл
   - Примените ко всем чатам или выборочно

## Примеры обоев
- `space_loop.mp4` — летающие звёзды и планеты
- `waves.mp4` — плавные волны цвета
- `matrix_rain.mp4` — цифровой дождь (в стиле "Матрицы")
